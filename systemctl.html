Systemctl

这是一个系统自带的命令。

  • start a service: systemctl start mysqld.service
  • stop a service: systemctl stop mysqld.service
  • set start when computer starts: systemctl enable mysqld.service
  • diable auto-start: systemctl disable mysqld.service
  • restart a service: systemctl restart mysqld.service
  • check the status of a service: systemctl status mysqld.service
  • see all the services that started: systemctl list-units --type=service
