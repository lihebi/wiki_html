Unicode

This page talks about my comprehension of Unicode use in python.

概述

  • Unicode是字符串，可以表示任意字符
  • 在文本和socket等中，使用的是字节串
  • 字节串--->Unicode: decode
  • Unicode--->字节串: encode
  • 编码方式有: utf8, gb2312, latin, ascii等。

python相关

a = u'xxxxx' #Unicode b = a.encode('utf8') #utf8 c = b.decode('utf8') a == c #True
